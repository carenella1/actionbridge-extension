# ActionBridge

Extract action items from any text on the web.

## How it works

1. Highlight text
2. Right-click
3. Select **Extract Action Items**

ActionBridge analyzes the text and displays detected tasks instantly.

## Install (Developer Mode)

1. Download this repository
2. Open Chrome
3. Go to `chrome://extensions`
4. Enable **Developer Mode**
5. Click **Load Unpacked**
6. Select the `extension` folder

Done.

## Example

Text:

Alex: I’ll send the updated proposal tomorrow.

ActionBridge:

Task: Send updated proposal
Owner: Alex
